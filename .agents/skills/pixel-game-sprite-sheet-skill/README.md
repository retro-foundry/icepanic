# Pixel Game Sprite Sheet Skill

A reusable skill for making game-ready retro pixel-art spritesheets with image generation and post-processing.

It focuses on:

- aligned animation sheets
- walk/idle/up/down rows
- preserving existing sprite-sheet layouts during repainting
- fixing common image-generation failures
- exporting cropped frames, GIF previews, and limited-palette PNGs

Start with `SKILL.md` for the instructions, `templates/prompts.md` for copy/paste prompt patterns, and `scripts/sprite_sheet_tools.py` for frame extraction and packing.
